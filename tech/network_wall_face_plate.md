# 벽의 LAN선 꽂이 정상화

인테리어 공사에서 마루에서 안방으로 가는 랜선을 4구멍 전화선 plate로 마감해 놓았다. 정상화 시켜 보자.

* 콘센트 brand: nano
* 안방은 RJ45 한 개면 될 듯
* 마루는 RJ45와 전화선을 연결해야 함
  * 전화선은 외부에서 인터넷 연결
  * RJ45는 마루에서 안방까지 연결

## 참고

* 랜용: [RJ45](https://en.wikipedia.org/wiki/Modular_connector#8P8C)
* 전화용: [RJ11](https://en.wikipedia.org/wiki/Registered_jack#RJ11)
